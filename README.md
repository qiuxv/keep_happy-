关于你的转专业的问题，我和另一位同学分开作答，希望能保持视角的独立性，给你更多的参考价值。在这里做出声明：我们的回答都基于我们过去的经历，具有不可避免的局限性。同时受限于我们的认识，客观上无法避免误解和不适合你或者现在时代的情况。故仅做参考，仍希望有借鉴的价值。

关于转专业的问题，我在你的这个阶段也有相同的困惑。我也是信息安全专业入学。我和另一位学长不同的是，我后来转入了软件工程专业（于大一）。我当时的主要考量是这样的：第一，同学的氛围让我感到不适，大家都在非常努力的卷绩点。一些老师也在热情地鼓励这个风气。而我不太喜欢。我不太喜欢念书。虽然我后来保研到南开，实际上我的绩点也不高。专业排名大概在20%.但是当时我也不知道软工的氛围如何，其实带着很大的盲目性。但是当时的内部竞争之激烈让我感到不适。第二，其实这个原因在我心中可能分量更高。我在很早的时候就参加了当时xie学长的ctf培训，后来也参加了一些相关的学习和竞赛。这让我感到非常困难而有压力，让我感到不适合这个专业。加上当时内部竞争氛围很浓，初入大学的焦虑和紧张。让我压力很大，情绪很糟。我希望打破这种局面，转专业主要是期待一个新环境，希望能够有所改变。

但是从现在回首去思考，从客观的角度来说。其实专业对于就业影响没有我当初想的那么大。计算机类、甚至电子信息类，甚至整个电类在就业上认可度都没那么大差别。软工的同学和信安的同学在就业的认可几乎没有差别，除非特殊的岗位，但是不论是实力还是运气都能轻松弥补掉这个差别。但是由于同学的主观影响，信安的同学做网安、运维的倾向要强一些，人也多一些。这个倾向也体现在升学上。信安的同学更多地去了（网络/信息/AI/...）安全院系和方向。

其二是课程区别。这一点很小。在这些很小的差别中，软工的课给我的感觉比较水。信安后面的课我没学，我转走了。这一点影响感觉也很小。
其三是升学上，和就业同样存在一个主观上的倾向。在绝大部分的情况下，计算机大类下的专业区别不那么大。如我的本科专业是软工，硕士方向是AI安全，学位是电子信息。

其三的氛围区别。我个人认为，信安在我当初的激烈的内部竞争情况和个别老师和领导脱不开关系。这也就导致了信安在我那年的保研难度很高。大量的加分和高绩点，如果以我在软工的绩点和加分，在信安名额也拿不到。但是在我后一届据我耳闻已经有所改善。至于你这一届，我不太清楚。近几年老师和领导的人员变动，在我看来，一切都朝着更好的方向发展。

从你的提问看来，你似乎很关心就业和升学问题。我的看法是，只要自己的思想不去限制自己，信安并没有比其他专业更加的局限，反而更加开阔。安全的未来没有网上说的那么好，但也没有那么糟糕。而且也不局限于安全。至于技术需要自学，我可以自认为很严谨地说，计算机课程在我校乃至中国高校的本科教育是很糟的。几乎没法学到什么很有用的知识，干货比较多的课存在，但是我觉得没有一半。而且也没有覆盖需要的知识。因此我觉得学计算机最重要的其实就是去自学，这有两点，一点是主动性，一点是自学能力。这都可以培养。
